Public/Private key cryptosystem:

TO BUILD:
	run makefile

TO EXECUTE:
	run the executable "p2" with one of the following command line arguments:
		-k generate keys to pubkey.txt and prikey.txt
		-e encrypt ptext.txt to ctext.txt
		-d decrypt ctext.xt to ptext.txt
